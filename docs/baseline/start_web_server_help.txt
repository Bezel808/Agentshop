usage: start_web_server.py [-h] [--port PORT] [--host HOST]
                           [--datasets-dir DATASETS_DIR] [--use-llamaindex]
                           [--simple-search] [--condition-file CONDITION_FILE]
                           [--max-pages MAX_PAGES]

启动 ACES-v2 Web 服务器

options:
  -h, --help            show this help message and exit
  --port, -p PORT       端口号（默认: 5000）
  --host HOST           监听地址（默认: 0.0.0.0，允许远程访问）
  --datasets-dir, -d DATASETS_DIR
                        数据集目录路径
  --use-llamaindex      使用 LlamaIndex RAG 检索（默认: True）
  --simple-search       使用简单文件名匹配（禁用 LlamaIndex）
  --condition-file, -C CONDITION_FILE
                        实验条件 YAML/JSON 文件路径（如
                        configs/experiments/example_price_anchoring.yaml）
  --max-pages, -n MAX_PAGES
                        每次最多展示页数（默认 5）
